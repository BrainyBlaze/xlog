\begin{abstract}
\noindent
We present xlog, a GPU-native logic programming language designed for unified symbolic reasoning. The language provides typed predicates, user-defined functions, modules, arithmetic, stratified aggregations, and integrity constraints, plus probabilistic facts and neural predicate declarations as first-class syntactic constructs. Programs compile to four reasoning backends on a single CUDA runtime: deterministic Datalog evaluation via semi-naive fixpoint, probabilistic inference via knowledge compilation (PIR $\to$ CNF $\to$ D4 $\to$ XGCF), SAT/MaxSAT verification, and differentiable neural-symbolic training---with zero host--device transfers in production paths. xlog is implemented in Rust with 22 custom CUDA kernel files (14.4K lines of device code) and provides zero-copy interoperability with PyTorch, JAX, and columnar analytics frameworks via DLPack and Arrow interfaces. Circuit caching across training iterations yields a measured $2.74\times$ end-to-end speedup (95\% CI: $[2.29, 3.18]$) on the MNIST addition benchmark.
\end{abstract}

# Language Completeness Showcase

This directory is the executable language-completeness suite. Each numbered
example has a `program.xlog`, an `expected.json` contract consumed by the
language-completeness validator, and short notes in `README.md`.

The older sibling directories under `examples/language-completeness/` remain focused
fixtures for individual feature tests. The showcase examples collect those
features into validator-owned programs with explicit expected diagnostics.

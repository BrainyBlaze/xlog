## Obtaining the data

The generated `data_*/` directories are **not tracked in this repository**: CLUTRR data
is licensed CC-BY-NC 4.0, which is incompatible with redistribution under this
repository's MIT/Apache-2.0 terms. To run this baseline, regenerate the splits with the
upstream generator (https://github.com/facebookresearch/clutrr) or copy the `data_*`
directories from the DeepProbLog project's CLUTRR experiment
(https://github.com/ML-KULeuven/deepproblog, `src/deepproblog/examples/CLUTRR/data/`),
placing them next to this file. Directory names encode the generator config hashes
expected by `clutrr.py`.

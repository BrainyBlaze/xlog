pub const WCOJ_SUPPORTED_KEY_TYPES: &[ScalarType];
Expand description

Scalar types that the WCOJ reference evaluator supports as join-key types. Membership is checked by analyze_typed when emitting Boundary::UnsupportedKeyType.

The set is intentionally narrow and not configurable in PR 2: only U32, U64, and Symbol are supported. Future PRs may widen the set as the reference evaluator and (later) GPU kernels grow type coverage; widening is a deliberate configuration change, not a parameter to this function.

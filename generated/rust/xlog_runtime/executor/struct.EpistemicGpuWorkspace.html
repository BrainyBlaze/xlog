Expand description

Device-resident buffers for epistemic Generate-Propagate-Test execution.

Fields§

§layout: EpistemicGpuWorkspaceLayout

Workspace layout used for allocation.

Candidate-assumption bitset buffer.

Candidate and accepted world-view bitset buffer.

§model_membership: TrackedCudaSlice<u8>

Per-model membership check buffer.

§rejection_reasons: TrackedCudaSlice<u32>

Structured rejection-reason code buffer.

§constraint_violation_index: TrackedCudaSlice<u32>

Per-candidate firing integrity-constraint index buffer. Parallel to rejection_reasons, sized layout.rejection_reason_slots. Holds the declaration-order index of the constraint that rejected a candidate, or the sentinel u32::MAX when no integrity constraint rejected it. The reason code in rejection_reasons is left at 6 for constraint violations; this buffer adds the constraint-specific detail.

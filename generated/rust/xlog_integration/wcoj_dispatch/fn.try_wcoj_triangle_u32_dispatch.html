Env-driven entry. Reads XLOG_USE_WCOJ_TRIANGLE_U32 and delegates to try_wcoj_triangle_u32_dispatch_with_gate.

Tests should prefer try_wcoj_triangle_u32_dispatch_with_gate to avoid races on the process-global env var; the env-driven form exists for production callers that want to opt in via configuration alone.
